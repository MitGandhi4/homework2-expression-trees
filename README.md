# Homework 2 — Expression Trees

## Description
This project implements an expression tree built from a postfix expression.
It supports tree traversals and evaluation of the expression.

## Files
- stack.py — Stack implementation
- expression_tree.py — Builds and traverses the expression tree
- evaluator.py — Evaluates postfix expressions
- test_cases.py — Test file for verifying correctness

## Features
- Build expression tree from postfix expression
- Preorder traversal
- Inorder traversal
- Postorder traversal
- Evaluate expression
